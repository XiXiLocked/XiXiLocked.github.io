一般是这么定义序偶的，先给2个$x,y$，然后就有了$( x,y )$。定义相等是这样的$( x,y ) = ( x' ,y' ) \Leftrightarrow x=x' ,y=y'$。

这里书上就说有个问题了，给了2个$x,y$ 就能做出序偶了，他是集合吗有公理支持他的存在吗？于是就有了这么一个蛋疼的习题，说：我们可以这么定义序偶$( x,y ) := \{ \{ x \} , \{ x,y \} \}$，$( x,y )$只是集合的缩写，而且集合本身有相等的定义，能够导出上面相等的条件，要验证一下。

$( \Leftarrow ) :x=x' ,y=y' \Rightarrow \{ x \} = \{ x' \} , \{ x,y \} = \{ x',y' \} \\ \Rightarrow \{ \{ x \} , \{ x,y \} \} = \{ \{ x' \} , \{ x' ,y' \} \} \\ \Rightarrow ( x,y ) = ( x' ,y' )$

$( \Rightarrow ) : ( x,y ) = ( x' ,y' ) \Leftrightarrow \{ \{ x \} , \{ x,y \} \} = \{ \{ x' \} , \{ x' ,y' \} \}$
分情况1.$x=y$，
$\{ \{ x \} , \{ x,y \} \} = \{ \{ x \} \} = \{ \{ x' \} , \{ x' ,y' \} \} \Rightarrow \{ x \} = \{ x' \} , \{ x \} = \{ x' ,y' \} \\ \Rightarrow x=x' =y=y' \Rightarrow x=x' ,y=y'$
2.$x \neq y$,
$( 1 ) \{ x \} = \{ x' \} , ( 2 ) \{ x \} = \{ x' ,y' \}$
并且
$( 3 ) \{ x,y \} = \{ x' ,y' \} , ( 4 ) \{ x,y \} = \{ x' \}$
于是有((1),(3)),((1),(4)),((2),(3)),((2),(4)) 4种组合，但是(4)本身或者((2),(3))都会导致$x=y$，与假设矛盾。所以考虑((1),(3)),$\{ x \} = \{ x' \} , \{ x,y \} = \{ x' ,y' \}$.
要不导致矛盾，只能$x=x' ,y=y'$ .
总之证起来虽然不难，但是分支不少挺麻烦。
然后书上又说，其实还有一种用集合的定义法$( x,y ) := \{ x, \{ x,y \} \}$，这个也可以作为序偶的定义，但是满足条件需要用到正则公理。 大概是这个样子(这个方向$( \Leftarrow)$的很显然就不写了，累。。。)：
$( \Rightarrow ) : ( x,y ) = ( x' ,y' ) \Leftrightarrow \{ x, \{ x,y \} \} = \{ x' , \{ x' ,y' \} \} \\ \Rightarrow \left\{\begin{array}{l} x=x' ( 1 )\\ x= \{ x' ,y' \} ( 2 ) \end{array}\right.$
并且$\left\{\begin{array}{l} \{ x,y \} =x' ( 3 )\\ \{ x,y \} = \{ x' ,y' \} ( 4 ) \end{array}\right.$
$( 1 ) , ( 3 ) \Rightarrow x= \{ x,y \} \Rightarrow x \in x= \{ x,y \} \\ ( 2 ) , ( 3 ) \Rightarrow x \in \{ x,y \} =x' ,x' \in \{ x' ,y' \} =x \\ ( 2 ) , ( 4 ) \Rightarrow x= \{ x' ,y' \} = \{ x,y \} \Rightarrow x \in x= \{ x,y \}$
上面都是被正则公理排除的情况,$A \in A$和$A_{1} \in A_{2} ,A_{2} \in A_{1}$都是被公理排除的。
$( 1 ) , ( 4 ) \Rightarrow \left\{\begin{array}{l} x=y \Rightarrow x=y=x' =y' \Rightarrow x=x' ,y=y'\\ x \neq y \Rightarrow y \neq x=x' \Rightarrow y=y' \Rightarrow x=x' ,y=y' \end{array}\right.$.完成/
